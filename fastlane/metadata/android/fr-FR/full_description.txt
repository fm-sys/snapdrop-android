Snapdrop pour Android est un client Android libre et open source pour partager des fichiers locaux https://snapdrop.net/.

Avez-vous aussi parfois le problème que vous avez besoin de transférer rapidement un fichier de votre téléphone vers votre PC?

USB? - Ancienne mode !
Bluetooth? - Beaucoup trop lourd et lent!
E-mail? - S'il vous plaît pas un autre e-mail que je m'écris à moi-même!

Snapdrop!

Snapdrop est une solution de partage de fichiers locaux qui fonctionne depuis votre navigateur. Un peu comme l'Apple Airdrop, mais pas seulement pour les appareils Apple. Windows, Linux, Android, IPhone, Mac - pas de problème!

Cependant, même si théoriquement cela fonctionnerait pleinement dans votre navigateur, vous allez adorer cette application si vous voulez utiliser Snapdrop plus souvent dans votre vie quotidienne. Grâce à une intégration parfaite dans le système d'exploitation Android, les fichiers sont envoyés encore plus rapidement. Directement à partir d'autres applications, vous pouvez sélectionner Snapdrop pour partager.

Grâce à sa simplicité radicale, "Snapdrop pour Android" facilite la vie quotidienne de centaines d'utilisateurs. En tant que projet open source, nous n'avons aucun intérêt commercial, mais nous voulons rendre le monde un peu meilleur. Rejoignez et convainquez-vous!

CODE SOURCE :
https://github.com/fm-sys/snapdrop-android

CONFIDENTIALITÉ:
Vos fichiers ne sont envoyés à aucuns serveurs. Les fichiers ne sont envoyés qu'entre les pairs de votre réseau local.

CRÉDIT:
L'application et son icône sont basées sur le projet Open Source Snapdrop qui est hébergé et maintenu par RobinLinus. Voir aussi https://www.github.com/robinlinus/snapdrop
